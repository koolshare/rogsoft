- Full Cone意味着更开放的网络，改善网络环境的同时也意味着更多的网络风险。

**反激活**
- 反激活可将激活码与本机解绑，解绑后该激活码可用于其他设备（或本机再次激活）；
- 注意：反激活次数用尽后，该激活码将绑定在最后一次激活的设备，直至到期；
- 如果你的反激活次数用光，可以通过购买延长授权来获得新的反激活次数；
- 试用授权/授权不支持反激活，购买授权（≥2年）支持按档位的反激活次数。
- 授权码反激活后并不会冻结使用时间，而是根据授权码到期日计算授权到期。

**试用规则**

- 试用期间/到期后可随时购买授权套餐，购买授权的权益将覆盖试用权益；
- 试用期间/到期后可以使用兑换码，兑换码兑换的授权类型将覆盖试用权益；

**兑换码**

- 兑换码分赠送/购买，前者用于推广，后者用于其它支付方式发码；
- 兑换码格式：fx_xxx，使用后会兑换为格式：fc_xxx的授权码；
- 赠送/购买类型兑换码可以兑换相应授权类型的授权码（fc_xxx）；
- 赠送授权可以覆盖试用授权，赠送授权会被购买授权覆盖；
- 授权天数多的兑换码会覆盖授权天数低的赠送授权，反之不会。
